# Flux de treball

Consulta [0-START-HERE.md](./0-START-HERE.md) abans de començar i [DEBUG-RUNBOOK.md](./DEBUG-RUNBOOK.md) quan hi hagi errors de Drive, media o persistència.

1. `npm.cmd run debug:status`
2. Canvi mínim.
3. `npm.cmd run verify:fast`
4. QA dirigida i captures de l'àrea afectada.
5. `npm.cmd run qa` abans de tancar un canvi ampli.

No publiquis Drive, importis JSON ni modifiquis permisos durant una inspecció sense autorització explícita.
